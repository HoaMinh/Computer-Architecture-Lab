Hướng dẫn ParaCache

Về trang chủ

Giới thiệu

ParaCache là trình mô phỏng cache để phân tích hit/miss theo các cấu hình: block size, associativity, replacement policy.

Hướng dẫn nhanh

  1. Chọn kích thước cache, block size, số dòng, mức liên kết (associativity).
  2. Nhập trace hoặc chương trình mẫu (địa chỉ truy cập).
  3. Run và quan sát thống kê hit, miss, miss rate.

Gợi ý phân tích

So sánh direct-mapped vs set-associative, quan sát ảnh hưởng block size lên spatial locality.
