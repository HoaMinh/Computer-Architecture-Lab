Hướng dẫn JsSPIM

Về trang chủ

Giới thiệu

JsSPIM (hoặc SPIM trên web) mô phỏng môi trường SPIM cho MIPS32. Hữu dụng để chạy file .asm, quan sát registers và memory.

Sử dụng

  1. Mở web JsSPIM
  2. Paste code .asm vào editor
  3. Run / Step / Reset
  4. Xem registers, stack, memory

Tips

  • Luôn khai báo .data và .text rõ ràng
  • Dùng syscall đúng mã
