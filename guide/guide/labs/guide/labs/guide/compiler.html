Hướng dẫn Compiler Explorer (Godbolt)

Về trang chủ

Giới thiệu

Compiler Explorer cho phép nhập mã C/C++ ở khung bên trái, và xem mã Assembly do compiler sinh ra ở khung phải. Có thể chọn compiler, kiến trúc, và flags.

Các bước cơ bản

  1. Mở godbolt.org
  2. Nhập C code
  3. Chọn compiler (ví dụ: gcc, clang) và option (-O0, -O2,...)
  4. Quan sát assembly, tìm phần code tương ứng

Ứng dụng trong lớp

  • Minh họa cách compiler tối ưu vòng lặp, inline, tail-call.
