Về trang chủ

Giới thiệu

ParaCache là trình mô phỏng cache để phân tích hit/miss theo các cấu hình: block size, associativity, replacement policy.

Hướng dẫn nhanh

  1. Chọn kích thước cache, block size, số dòng, mức liên kết (associativity).
  2. Nhập trace hoặc chương trình mẫu (địa chỉ truy cập).
  3. Run và quan sát thống kê hit, miss, miss rate.

Gợi ý phân tích

So sánh direct-mapped vs set-associative, quan sát ảnh hưởng block size lên spatial locality.

Mục tiêu

Hiểu tác động của cấu hình cache lên hit/miss và thời gian truy xuất trung bình (AMAT).

Bài 1

Cho trace sau (địa chỉ hex): 0x00,0x04,0x08,0x0C,0x00,0x04,0x10,... Thử cấu hình:

  • Cache size = 16 bytes; block size = 4 bytes; direct-mapped
  • Ghi lại hit/miss

Bài 2

Thay associativity = 2-way; so sánh miss rate; phân tích nguyên nhân.

Báo cáo

Nộp bảng kết quả, biểu đồ so sánh, và kết luận ngắn.
