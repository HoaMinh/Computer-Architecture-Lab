Bài thực hành JsSPIM

Về trang chủ

Mục tiêu

Luyện kỹ năng viết chương trình Assembly MIPS nhỏ: IO, vòng lặp, thao tác mảng và gọi hàm.

Bài 1: In dãy Fibonacci n số

Viết chương trình in n số Fibonacci (n ≤ 20). Input n đặt sẵn trong .data hoặc dùng syscall để nhập.

Bài 2: Sắp xếp mảng

Viết chương trình sắp xếp mảng (bubble sort) và in kết quả.

Nộp bài

Nộp file .asm và màn hình chạy (screenshot). Ghi chú thời gian thực thi (nếu cần) và phân tích ngắn về registers dùng.
